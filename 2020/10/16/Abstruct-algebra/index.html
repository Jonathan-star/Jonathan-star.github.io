Quotient Define & Examples

In the algebra, many system has its quotient system, like quotient group,quotient set, and so on.

Quotient Group

The quotient needs to define a relation just like $R$, to simplify the argument, I use group to discuss.

with the defined relationship, we have a Left Coset , the Left Cosets build up the quotient group, we can present each elements by the Representative of equivalence class, we write them as $\bar a, \bar b \dots$

$\bar a$ Represent each elements that equal to $a$ under the relationship ${R}$

the simplest example is the set $\mathbb{Z}$ under the calculate +, and $\mathbb{Z}/m\mathbb{Z}$, the quotient group of it.

the elements in $\mathbb{Z}/m\mathbb{Z} $, just like $\bar 1, \bar 2, \cdots, \overline{m-1}$, in the +, the equation is presented as $ \text{if } a - b = i$, so $a \& b $ is in the set $\overline{i}$, in the $\times$ is the $ab^{-1} \in \overline{i}$.
